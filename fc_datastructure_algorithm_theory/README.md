fast-campus 알고리즘 온라인 강의를 들으면서 작성한 알고리즘 코드들입니다.

- shift+enter -> 현재 줄에 있는 코드 실행

# 자료구조

- 배열, 큐, 스택, 링크드리스트, 해쉬테이블, 트리, 이진 탐색 트리, 힙

# 알고리즘

- 정렬
- 재귀호출, 동적 계획법, 분할 정복, 탐욕 알고리즘, 백트래킹
- 탐색
- 그래프
  - DFS, BFS
  - 다익스트라
  - 신장트리
    - 크루스칼
    - 프림

- 페캠에서 나와 있는 로직으로 한 번 구현해 볼 것 + 이해
